# ✨ Gesture Particle Universe

An interactive **3D particle simulation** that uses real-time **hand gesture recognition** to control and transform a dynamic universe. Built with **Three.js** and **MediaPipe Hands**, this project turns simple hand movements into beautiful cosmic animations.

---

## 🚀 Live Interaction Features

🖐️ Real-time hand tracking using webcam  
🌌 3D particle universe (20,000+ particles)  
💫 Gesture-based shape transformations  
🎨 Smooth animated transitions  
📷 Live camera preview window  
⚡ High-performance WebGL rendering

---

## 🧠 Gesture Controls

| Gesture | Action |
|--------|--------|
| ✊ Fist | 💥 Fireworks burst explosion |
| ☝️ 1 finger | ❤️ Heart nebula |
| ✌️ 2 fingers | 🌸 Flower spiral |
| 🤟 3 fingers | 🪐 Saturn ring |
| ✋ 4+ fingers | 🌌 Solar system |
| 👆 Hand tilt | Rotate universe |

---

## 🛠️ Tech Stack

- HTML5 / CSS3 / JavaScript
- 🎮 Three.js (3D rendering)
- 🤖 MediaPipe Hands (gesture detection)
- 📷 WebRTC Camera API
